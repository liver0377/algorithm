**题解**

这里没什么特殊的方法，具体思路就是当扫描到一个元素的下标为k, 且从0~k的区间的最大元素的值为k时，那么就代表该区间可以被排序形成一个块

这里注意一下，对于一个长度为k的数组来说，分成的块的个数最多也就是k个，扫描一遍得到的res肯定也不会超过k

```cpp
int maxChunksToSorted(vector<int>& arr) {
      int res = 0;
      int m= arr[0];
      for (int i = 0; i < arr.size(); i++) {
          m = max(m, arr[i]);
          if (i == m) {
              res ++;
          }
      } 
      return res;
    }
```
