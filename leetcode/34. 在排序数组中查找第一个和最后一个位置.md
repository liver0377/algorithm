





**代码实现**

- 使用`equal_range`

```cc
vector<int> searchRange(vector<int>& nums, int target) {
        auto [x, y] = equal_range(nums.begin(), nums.end(), target);
        
        if (x == nums.end() || *x != target) return {-1, -1};
        return {static_cast<int>(x - nums.begin()), static_cast<int>(y - nums.begin() - 1)};
}
```
