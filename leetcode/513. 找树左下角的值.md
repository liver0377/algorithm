**解题思路**

- 此题使用层序遍历求解
- 在层序遍历时, 每次按照从右向左的顺序加入节点
- 那么左下角的节点一定是最后一个入队以及最后一个出队的
- 在循环中, 每次弹出一个节点, 然后加入它的孩子节点即可



**代码实现**

```py
class Solution:
    def findBottomLeftValue(self, root: Optional[TreeNode]) -> int:
        q = deque([root])

        while q:
            node = q.popleft()
            if node.right:
                q.append(node.right)
            if node.left:
                q.append(node.left)
        
        return node.val
```
